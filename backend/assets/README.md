# Certificate Template Setup

Place your certificate template PNG image here as `certificate-template.png`

## Template Requirements:
- **Size**: 800x600 pixels (landscape orientation)
- **Format**: PNG with transparent background (optional)
- **Design**: Include placeholder areas for text that will be overlaid

## Text Overlay Positions:
The system will automatically overlay text at these positions:
- **Title**: "CERTIFICATE OF COMPLETION" (centered, top)
- **User Name**: Large, prominent text (centered, middle-top)
- **Course Name**: Course title in quotes (centered)
- **Project Name**: Project title (centered)
- **Date**: Completion date (centered)
- **Certificate ID**: Small text at bottom (centered)

## Design Tips:
- Leave space in the center for the main text
- Use a clean, professional design
- Consider using a border/frame
- Add subtle background patterns or gradients
- Include your platform branding

The text will be overlaid on top of your template image, so design accordingly!
